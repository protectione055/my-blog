> **构建理念** · 让内容保持专注、画面保持沉静、体验保持顺滑。

欢迎来到全新版本的博客。主题完全采用官方 Blowfish 风格：流动的渐变背景、作者简介与社交入口置于视线中心，并以卡片布局呈现最近的文章。为了让不同语言的读者都能快速进入状态，站点会自动识别语言并切换到对应的内容树。

- Hugo Extended + Blowfish 2，具备暗色默认外观与自动切换能力
- Tailwind 风格的组件与 smart TOC，阅读时可随手折叠目录
- 文章默认展示背景封面、阅读时间、系列导航与分享按钮
- 借助 `npm run dev` 可本地热更新，`npm run build` 输出可部署静态文件
